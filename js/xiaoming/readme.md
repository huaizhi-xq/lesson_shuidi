## JS 面向对象
  - JS数据类型
    {} 对象 Object
    [] 数组
    字符串 "lgm" String
    数值型 Number 21
    布尔值类型 Boolean
  - JS中面向对象有俩种表达方式
    1. class 方式来构造
    抽象的类，以及可使用的对象的概念 constructor type n
    2. 对象字面量的方式来构建 { } 描述性
    属性+方法的复杂数据类型
    xm.age xm.sendFlower(xh);
  - 代理模式 Proxy
    小明 -> 小红 简陋
    小明 -> 小刘 -> 小红 复杂？
    不同的对象（小红，小刘）,只要实现同样的方法(receiveFlower),交换使用,面向接口的编程，代理模式
